# Прогнозирование сердечно-сосудистых заболеваний

Описание проекта

В данном проекте решается задача прогнозирования сердечно-сосудистых заболеваний с использованием методов машинного обучения и нейронных сетей.

Цель проекта — построить модели, способные предсказывать наличие заболевания на основе медицинских показателей пациента.

Данные

Датасет содержит медицинские данные пациентов, включая:

* возраст (age)
* пол (sex)
* артериальное давление (resting_blood_pressure)
* уровень холестерина (serum_cholestoral)
* уровень сахара в крови (fasting_blood_sugar)
* максимальная частота сердечных сокращений
* результаты ЭКГ
* и другие признаки

Размер датасета:

* обучающая выборка: 600 000 записей
* тестовая выборка: 400 000 записей

Анализ данных

В рамках проекта были выполнены:

* разведочный анализ данных (EDA)
* построение корреляционной матрицы
* scatter matrix (диаграммы рассеяния)
* анализ выбросов (boxplot)

Используемые модели

В проекте были обучены и сравнены следующие модели:

* Logistic Regression
* Random Forest
* Нейронная сеть (PyTorch)

 Результаты

| Модель              | ROC-AUC |
| ------------------- | ------- |
| Random Forest       | ~0.96   |
| Нейронная сеть      | ~0.958  |
| Logistic Regression | ~0.948  |

Лучшую точность показала модель Random Forest.

Нейронная сеть

Реализована полносвязная нейронная сеть с:

* несколькими слоями
* функцией активации ReLU
* Dropout для регуляризации
